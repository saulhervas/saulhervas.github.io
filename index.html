Aplicación de Colección de Bonsáis

Esta aplicación en Kotlin está diseñada para los amantes de los bonsáis, permitiendo a los usuarios gestionar y organizar su colección personal de estas hermosas miniaturas de árboles. Integra tecnologías modernas para ofrecer una experiencia fluida y eficiente.


  • Registro Completo de Bonsáis: Permite almacenar detalles como el nombre, la fecha de adquisición, la fecha del último trasplante y la fecha del próximo. Además, los usuarios pueden guardar imágenes de cada bonsái para llevar un registro visual de su evolución.
  • Gestión de Datos con Firebase: Los datos y las imágenes de los bonsáis se almacenan de forma segura en Firebase, asegurando que la información esté accesible en cualquier momento y desde cualquier dispositivo.
  • Navegación Fluida: Implementación de Navigation para facilitar la navegación entre las distintas secciones de la aplicación, proporcionando una experiencia de usuario intuitiva.
  • Optimización con Corrutinas: Uso de corrutinas para manejar de manera eficiente las operaciones asincrónicas, asegurando que la aplicación se mantenga rápida y receptiva.
  • Inyección de Dependencias con Hilt: Implementación de Hilt para una gestión simplificada y eficiente de las dependencias, mejorando la escalabilidad y el mantenimiento de la aplicación.

Aplicación de sumas y restas para niños

La Aplicación de Sumas y Restas para Niños será una herramienta interactiva, lúdica y educativa diseñada para que los niños practiquen operaciones básicas de matemáticas (sumas y restas) de una manera divertida y sencilla. La aplicación estará desarrollada en Kotlin para dispositivos Android y tendrá una interfaz colorida y amigable, con elementos grandes y fáciles de tocar para adaptarse a las habilidades motoras de los niños.


  • Generador Aleatorio de Problemas: Cada vez que el niño abre la aplicación, se generan problemas matemáticos aleatorios para resolver.
  • Retroalimentación Inmediata: Al responder una operación, el niño recibirá retroalimentación instantánea (splash) para indicar si la respuesta es correcta o incorrecta.
  • Operaciones Básicas: Módulos separados para sumas y restas.

Aplicación para calcular el área

Esta aplicación en Kotlin está diseñada para calcular el área de tres figuras geométricas básicas: círculo, cuadrado y triángulo. Los usuarios pueden ingresar las dimensiones necesarias para cada figura y obtener rápidamente el área correspondiente.


  • Círculo: Introduce el radio para calcular el área utilizando la fórmula π × radio².
  • Cuadrado: Introduce el lado para calcular el área con la fórmula lado².
  • Triángulo: Introduce la base y la altura para calcular el área usando la fórmula (base × altura)/ 2.

Aplicación para agregar usuarios

Esta aplicación en Kotlin está diseñada para gestionar usuarios de manera eficiente y moderna. Utilizando Room para el manejo de bases de datos, la aplicación permite almacenar y gestionar información de usuarios de forma robusta y segura.

He utilizado corrutinas para realizar operaciones de base de datos y actualizaciones en segundo plano, garantizando un rendimiento fluido y una experiencia de usuario sin interrupciones. La interfaz de usuario se maneja mediante RecyclerView, ofreciendo una visualización dinámica y eficiente de la lista de usuarios.

La navegación entre pantallas se facilita con Navigation, permitiendo a los usuarios añadir nuevos registros y editar los datos existentes de manera intuitiva y estructurada.

Aplicación para búsqueda equipos de fútbol

Ésta es una aplicación móvil ideal para los aficionados al fútbol que desean descubrir información esencial sobre equipos de fútbol de manera rápida y sencilla. Con un diseño basado en MVVM (Model-View-ViewModel), la aplicación ofrece una experiencia de usuario eficiente y organizada.


  • Búsqueda Rápida de Equipos: Encuentra equipos de fútbol de todo el mundo utilizando una interfaz intuitiva.
  • Información Clave: Cada equipo se presenta con su nombre, el nombre del estadio, una foto representativa y el año de fundación.
  • Navegación Ágil: Navega sin esfuerzo entre diferentes equipos con una experiencia fluida proporcionada por el RecyclerView y la navegación optimizada.
  • Consumo de Datos: La aplicación utiliza Retrofit para consumir datos de una API de fútbol, mientras que las corrutinas manejan operaciones de red de manera asíncrona para una respuesta rápida.
